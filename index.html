SKILLS

  • Redux를 활용해 페이지별 상태를 관리하고 React Hook을 활용해 팝업, 툴팁 등 공통 UI 기능의 상태와 동작 로직을 구현했습니다.
  • 마켓 정책을 Configuration 객체로 분리하고 컴포넌트는 설정에 따라 렌더링만 수행하도록 설계했습니다.
  • AG-Grid를 활용해 대량의 상품/주문 데이터를 정렬, 필터링, 컬럼 관리 기능을 제공하는 공통 리스트 컴포넌트를 구현했습니다.
  • Atomic Design과 Storybook을 적용해 공통 컴포넌트를 관리했습니다.
  • ESLint, Prettier를 적용해 팀 내 코딩 컨벤션을 수립하고 코드 품질을 관리했습니다.
  • 백오피스에서는 디자인 리소스 없이 일관된 UI를 빠르게 구현하기 위해 MUI를 활용했습니다.
  • Node.js와 Express 기반으로 쇼핑몰 통합관리 서비스의 마켓 정책 변경에 따른 기능 개선 및 유지보수를 담당했습니다.
  • 신규 판매 채널의 Open API 분석해 상품, 주문, 배송 데이터를 시스템과 연동하여 내부 서비스 구조에 맞게 표준화하는 API를 구현했습니다.
  • MSA 환경에서 셀러 인증, 데이터 수집, 동기화, 배송 처리를 위한 REST API를 설계하고 YAML 워크플로우를 통해 마켓별 연동 규칙을 구현했습니다.
  • Open API에서 제공하지 않는 데이터는 스크래핑 기반 데이터 수집으로 일관된 데이터 처리가 가능하게 했습니다.
  • Swagger UI를 활용해 API 명세를 관리했습니다.
  • 라우팅 구조, 서버 사이드 렌더링, API Route를 활용해 초기 로딩과 검색 노출을 고려한 웹 서비스를 구현할 수 있습니다.
  • 전역 상태는 Redux로 관리하고 CRUD 중심 조회 데이터는 SWR로 분리해 상태 관리 부담을 줄였습니다.
  • MVC 구조를 바탕으로 인증, 데이터 모델링, 관리자 기능을 구성할 수 있습니다.
  • Blade 템플릿과 Laravel API를 연동해 서비스 화면과 백오피스 기능을 개발할 수 있습니다.
  • Vue 컴포넌트 구조와 상태 흐름을 이해하고 화면 단위 유지보수를 고려해 SPA 기능을 구현할 수 있습니다.
  • API 연동, 화면 상태 처리, 운영 이슈 대응을 통해 서비스 화면을 안정적으로 관리할 수 있습니다.

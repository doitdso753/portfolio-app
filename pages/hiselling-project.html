쇼핑몰통합관리 서비스

담당업무

  • 쇼핑몰 통합관리 화면 개발 및 공통 컴포넌트 설계 - React, Redux
  • 상품, 주문, 배송 관리 API 설계 및 개발 - Node.js
  • 관리자 사이트 구축 및 기능 개발 - Next.js, React

시스템 구성도

UI Service(React)와 Backend/Core Service(Node.js)로 구성된 쇼핑몰 통합관리 서비스의 개발 및 운영을 담당했습니다.
사용자는 UI Service를 통해 상품, 주문, 배송 관리 기능을 이용하며, UI Bridge가 Backend Service와 연동하여 비즈니스 기능을 제공합니다. Backend Service는 도메인별 API로 구성해 기능별 책임을 분리하고 Kafka 메시지 큐로 Backend Service 작업을 비동기로 처리합니다. Next.js 기반 Admin Service에서 마켓 기초 데이터 등 운영 기능을 관리할 수 있도록 구현했습니다. Docker/Kubernetes 컨테이너 환경에서 서비스를 운영했습니다.

서비스 화면

여러 온라인 쇼핑몰의 상품, 주문, 배송 등 운영 업무를 통합 관리할 수 있는 솔루션입니다. 상품, 주문, 배송 기능을 독립 API로 구성한 MSA 기반 구조입니다. 마켓 별로 컴포넌트를 관리합니다. 상품 등록, 주문 수집, 배송 관리 등의 기능을 제공합니다.

백오피스 화면

관리자 페이지 화면은 대시보드, 상품 관리, 주문 관리, 고객 관리 등으로 구성되어 있습니다. 마켓 기초 데이터 등 운영 기능을 관리할 수 있도록 구현했습니다.

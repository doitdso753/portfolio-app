시스템 구성도

Laravel 기반 API 서버로 구성된 웹 에디터 서비스의 개발 및 운영을 담당했습니다. Service API가 이미지 처리 및 비즈니스 로직을 담당하도록 구현했습니다. Python 기반 Capture API와 Image Thumbnail API를 구축하여 이미지 캡처, 썸네일 생성 등 전처리 기능을 제공했으며, AI API 연동을 통해 상품 특성에 맞는 상세페이지 문구 자동 생성 기능을 개발했습니다. 관리자 사이트는 Laravel 기반으로 콘텐츠 관리 기능을 개발했습니다.

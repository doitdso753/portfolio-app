주요 경험

고객 피드백과 운영 과정에서 발생한 문제를 분석하고 기능 개선을 수행했습니다. API 요청-응답 속도 저하 원인을 분석해 글로벌 미들웨어 구조를 개선한 결과 API 처리 속도를 약 40% 향상시켰고 이미지 품질 문제는 Playwright 기반 이미지 추출 서버를 구축해 관련 고객 문의를 약 66% 감소시켰습니다. 자동저장 기능을 추가해 데이터 유실 관련 고객 문의를 약 90% 감소시켰습니다.

서비스 구경하기

시스템 구성도

Laravel 기반 API 서버로 구성된 웹 에디터 서비스의 개발 및 운영을 담당했습니다. Service API가 이미지 처리 및 비즈니스 로직을 담당하도록 구현했습니다. Python 기반 Capture API와 Image Thumbnail API를 구축해 이미지 캡처, 썸네일 생성 등 전처리 기능을 제공했으며, AI API 연동을 통해 상품 특성에 맞는 상세페이지 문구 자동 생성 기능을 개발했습니다. 관리자 사이트는 Laravel 기반으로 콘텐츠 관리 기능을 개발했습니다.

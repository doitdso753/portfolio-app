적립&매장 마케팅 서비스

담당업무

  • Android 하이브리드 모바일 앱 개발
  • Vue.js 기반 화면 개발
  • KCP 결제 API 연동
  • 매장용 POS 앱 개발
  • 관리자 사이트 기능 개발
  • 기능 추가에 따른 DB 스키마 설계

기술스택

  • Frontend: Vue.js, JavaScript, Android, C# Windows Forms, Socket.io
  • Backend: PHP Laravel
  • Database: MySQL
  • Tools/Infra: Linux, Ncloud Object Storage, Git, GitLab, Notion, IntelliJ

모바일 앱 화면

고객은 모바일 앱에서 매장별 적립 상태와 쿠폰을 확인하고 매장에서 제공하는 혜택을 확인할 수 있습니다.
안드로이드 네이티브 카메라와 WebView를 연동해 QR 스캔 기능을 구현하고 Socket 통신을 통해 POS에서 고객 정보를 실시간으로 조회할 수 있습니다.

관리자 사이트 화면

또와요 서비스에 계약한 사장님께 도장/포인트 적립 관리, 쿠폰 관리, 고객 관리를 위해 제공하는 관리자 사이트입니다.

POS 앱 화면

사장님이 매장에서 QR 코드나 핸드폰 번호만으로 고객에게 쉽게 적립, 쿠폰/선물 사용할 수 있도록 제공하는 POS 앱입니다.

시스템 구성도

고객용 모바일 앱, 매장 POS 앱, 관리자 사이트로 구성된 적립&매장 마케팅 서비스입니다.
모바일 앱과 POS 앱은 Service API를 통해 회원, 적립, 쿠폰, 선물 데이터를 처리하도록 설계했으며, 결제 기능은 별도 Pay API로 분리해 PG사 연동했습니다.
관리자 사이트는 Laravel 기반으로 구축하여 매장, 쿠폰, 이벤트, 회원 정보를 통합 관리할 수 있도록 구현했습니다.
서비스 데이터는 MySQL에 저장하고 이미지 및 첨부 파일은 Object Storage에 분리 저장했습니다.
